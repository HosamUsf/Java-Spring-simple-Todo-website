# Java Spring simple todo app

## Initial Implementation
As a user you would like to have application to mange your todos 

### The user can :
* login 
* add todo
* update todo
*  delete todo



## Technologies Used
* **Java 17**
* **Maven**
* **Spring Boot 3**
* **Spring Data JPA**
* * **Spring Secuirty **
* ** H2 database  **
